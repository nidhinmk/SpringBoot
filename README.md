# Spring Boot Learning Projects – Summary

This repository contains a structured set of Spring Boot projects, progressing from core concepts to advanced enterprise features. Each module is focused, hands-on, and demonstrates a specific capability of the Spring ecosystem.


## Project Overview (Concise Explanations)

01 – Spring Boot Overview

Introduction to Spring Boot fundamentals, application structure, auto-configuration, and how a Spring Boot app starts and runs.

02 – Spring Boot Spring Core

Covers core Spring concepts: IoC container, Beans, Dependency Injection, scopes, and lifecycle management.

03 – Spring Boot Hibernate JPA CRUD

Hands-on implementation of Hibernate with Spring Data JPA, performing full CRUD operations with entity mapping.

04 – Spring Boot REST CRUD

Builds RESTful APIs using Spring Web, including endpoints for Create/Read/Update/Delete with JSON responses.

05 – Spring Boot REST Security

Introduces Spring Security basics: authentication, authorization, and securing REST endpoints.

06 – Spring Boot Spring MVC

Explains the Spring MVC architecture including controllers, routing, model-view handling, and serving dynamic content.

07 – Spring Boot Spring MVC CRUD

A complete CRUD application using Spring MVC, form handling, validation, and server-side rendering.

08 – Spring Boot Spring MVC Security

Applies Spring Security to MVC applications, securing pages, login workflows, and route protection.

09 – Spring Boot JPA Advanced Mappings

Advanced Hibernate/JPA mappings: one-to-one, one-to-many, many-to-many, join tables, cascading, and fetching strategies.

10 – Spring Boot AOP

Demonstrates Aspect-Oriented Programming: logging, cross-cutting concerns, custom annotations, and method intercepting.
